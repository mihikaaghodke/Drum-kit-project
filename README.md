🥁 Drum Kit Project
This is a fun, interactive drum kit built with HTML, CSS, and JavaScript. Users can click buttons or press keys to play different drum sounds.

🔧 Features
- Responsive layout
- Keypress and click support
- Custom drum sounds
- Smooth button animations
